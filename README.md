ğŸ§  Working Principle

The temperature sensor continuously monitors environmental temperature.

When the temperature exceeds a predefined threshold, the system detects a fire hazard.

The Raspberry Pi Pico W processes the sensor data using MicroPython.

A relay module is triggered to activate the water pump.

The rover moves toward the heat source and extinguishes the fire automatically.

Real-time status can be monitored via a Wi-Fi-based web interface.
